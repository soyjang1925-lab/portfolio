UX/UI Design 2024

Fintech Banking App

개인 자산 관리를 위한 핀테크 모바일 앱의 전체 UX/UI 리디자인 프로젝트. 복잡한 금융 데이터를 직관적이고 신뢰감 있는 인터페이스로 재해석했습니다. 사용성 테스트와 데이터 분석을 통해 사용자 이탈률 40% 감소를 달성했습니다.

Overview

Problem &
Challenge

기존 FinPay 앱은 기능은 충분했지만 정보 구조가 복잡하고 시각적 위계가 불명확해 사용자들이 핵심 기능을 찾기 어려워했습니다. 월평균 이탈률이 38%에 달했으며, 앱스토어 평점은 2.9점에 머물렀습니다.

사용자 인터뷰와 히트맵 분석을 통해 세 가지 핵심 문제를 발견했습니다: 과도한 정보 밀도, 불명확한 CTA 버튼 위계, 그리고 신뢰감 부족한 색상 시스템이었습니다.

저는 리드 디자이너로서 UX 리서치부터 최종 UI 산출물까지 전 과정을 주도했습니다.

"복잡한 금융 정보를 단순하게 — 신뢰와 명료함이 공존하는 디자인"

Process

Design
Thinking

01

Research & Discovery

기존 사용자 20명을 대상으로 심층 인터뷰를 진행하고, GA4와 Hotjar를 통해 실제 사용 패턴을 분석했습니다. 경쟁사 8곳의 벤치마킹으로 핵심 개선 기회를 도출했습니다.

User Interview Hotjar GA4 Competitive Analysis
02

Define & Synthesize

어피니티 다이어그램으로 인사이트를 클러스터링하고 3개의 핵심 페르소나를 정의했습니다. 사용자 여정 맵(User Journey Map)을 통해 주요 페인 포인트와 기회 영역을 시각화했습니다.

Affinity Map Persona Journey Map FigJam
03

Wireframe & IA

정보 구조(IA)를 재설계하고 로우파이 와이어프레임 48개를 제작했습니다. 내부 팀과 3회의 리뷰 세션을 거쳐 핵심 플로우를 확정했습니다.

Card Sorting IA Design Wireframe Figma
04

UI Design & System

새로운 디자인 시스템을 구축하고 하이파이 UI를 제작했습니다. 컴포넌트 라이브러리 120개 이상, 다크/라이트 모드 지원, WCAG 2.1 AA 접근성 기준을 충족했습니다.

Design System Auto Layout Variables Zeplin
05

Prototype & Test

Protopie로 고해상도 인터랙티브 프로토타입을 제작하고, 사용자 15명을 대상으로 모더레이트 사용성 테스트를 진행했습니다. 태스크 완료율이 기존 62%에서 91%로 향상되었습니다.

Protopie Usability Test A/B Test Maze

Results

Impact

40%
이탈률 감소
월평균 38% 이탈률이 리디자인 후 23%로 감소. 온보딩 플로우 개선이 핵심 요인.

+29%
전환율 향상
주요 금융 상품 가입 전환율 29% 향상. 개선된 CTA 위계와 신뢰감 있는 UI가 기여.

"디자인 퀄리티는 물론 UX 논리가 탄탄해서 개발팀과의 협업이 매우 원활했습니다. 단순히 예쁜 디자인이 아니라 비즈니스 목표와 사용자 니즈를 모두 충족시킨 결과물이었습니다."
